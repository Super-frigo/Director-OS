# Color Library

Status: Planned

## Purpose

Color Library 存储色彩方案与色彩心理学知识。

## Planned Categories

- Color Psychology — 每种颜色的情绪与文化含义
- Color Palettes — 经典配色方案(互补/类似/三角/分裂互补)
- Film Color Grading Reference — 经典影片的色彩风格
- Genre Color Codes — 类型片的色彩惯例(冷色科幻/暖色浪漫/高饱和奇幻)
- Skin Tone Lighting — 不同肤色在不同光线下的表现

## Example Entry

```yaml
- palette: "Teal & Orange"
  psychology:
    teal: "Cold, technological, distance"
    orange: "Warmth, humanity, intimacy"
  usage: "Blockbuster default. Skin tones pop against teal backgrounds."
  example_films:
    - "Transformers series"
    - "The Matrix Reloaded"
    - "Mad Max: Fury Road (variation)"
  alternatives:
    - "Green & Magenta (more natural)"
    - "Blue & Gold (more classical)"
```
