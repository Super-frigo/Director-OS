## Purpose

Stores narrative structure knowledge — act structures, beat types, character arc patterns, conflict types, and worldbuilding frameworks. Queried by Story Engine during PLANNING stage.

## Worldbuilding

The worldbuilding entry provides a complete framework for constructing narratively-consistent worlds:
- Core setting definition (name, era, conflict, tone)
- Physical laws and constraints
- Geography and environment
- Social structure and culture
- Visual anchors and style binding
- Narrative rules and questions
