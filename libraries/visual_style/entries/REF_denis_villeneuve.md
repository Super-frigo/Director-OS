# Denis Villeneuve 风格拆解 (Original Reference)

> Imported from AI-Director-OS/stylebook/denis-villeneuve.md

## 视觉 DNA

### 缓慢推进
- 镜头运动本身就是情绪
- 节奏来自不愿移动
- 留足够长的时间让观众进入画面

### 留白叙事
- 画面中大面积负空间
- 信息不在于画面中有什么，而在于没有什么
- 对白极简

### 环境压倒人物
- 角色被放在画面 1/3 以下的区域
- 自然/建筑不可动摇
- 存在主义的视觉表达
