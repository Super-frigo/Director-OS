- 材质质感

## 色彩体系
| 情绪 | 主色调 |
|------|--------|
| 温暖/家庭 | 暖橙 + 金琥珀 |
| 孤独/沉思 | 深蓝 + 淡紫 |
| 冒险/活力 | 高饱和绿 + 明黄 |
| 回忆/伤感 | 去饱和暖调 |
