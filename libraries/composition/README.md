# Composition Library

Status: Planned

## Purpose

Composition Library 存储构图法则与应用场景,与 SHOT_GRAMMAR.md 中的 CompositionRule 对应。

## Entries

- Rule of Thirds — 三分法基础应用
- Golden Ratio — 黄金比例构图
- Symmetry — 对称构图与叙事意图
- Frame Within Frame — 框中框的层次感
- Leading Lines — 引导线方向与情绪
- Negative Space — 留白的叙事力量
- Dynamic Symmetry — 动态对称与网格系统

## Example Entry

```yaml
- name: "Frame Within Frame"
  effect: "Creates voyeuristic feel, emotional distance"
  technique: "Use doorways, windows, arches, foreground elements"
  best_for:
    - Character isolation
    - Surveillance or hiding themes
    - Transition between worlds
  example_films:
    - "The Godfather (doorway framing)"
    - "Grand Budapest Hotel (window framing)"
  notes: "Each frame is a layer of separation from the truth."
```
