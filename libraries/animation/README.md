# Animation Library

Status: Planned

## Purpose

Animation Library 存储动画风格与技术参考,供动画类项目使用。

## Planned Categories

- 2D Animation Styles — 手绘/Flash/纸片动画/水墨
- 3D Animation Styles — 写实/风格化/三渲二/低面数
- Stop Motion Styles — 粘土/定格/物体动画
- Motion Graphics — 动态图形/数据可视化/UI动画
- Visual Effects Animation — 粒子/流体/布料模拟
- Animation Principles — 12条动画原则与参考
- Frame Rate Reference — 不同风格的最佳帧率
