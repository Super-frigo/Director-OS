# Advertising Library

Status: Planned

## Purpose

Advertising Library 存储商业广告的类型、格式与创意思路。

## Planned Categories

- Ad Types — 品牌故事/产品展示/情感共鸣/社会公益
- Format Reference — 15s/30s/60s 不同时长的叙事结构
- Industry Conventions — 汽车/时尚/科技/食品/奢侈品
- Viral Formula Analysis — 病毒传播的视频结构
- Call to Action — 不同 CTA 类型的视觉处理
- Platform-Specific — 横屏(TV/YouTube) vs 竖屏(TikTok/Reels/Shorts)
