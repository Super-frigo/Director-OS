```

- 0:10-0:45: Verse 1 — character intro
- 0:45-1:15: Chorus — energetic visuals, fast cuts
- 1:15-1:45: Verse 2 — narrative development
- 1:45-2:15: Chorus 2 — build intensity
- 2:15-2:45: Bridge — visual break
- 2:45-3:15: Final Chorus — climax
```
