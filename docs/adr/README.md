## Template

Each ADR follows this structure:

- Context
- Decision
- Consequences
- Scope
- Non-Goals
- Alternatives Considered
- Compliance
- Design Principle

## Rules

- One ADR addresses exactly one architectural concern.
- ADRs are immutable once Accepted. Superseded ADRs are explicitly replaced by a newer ADR.
