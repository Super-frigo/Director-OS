# 1. Overview

Project is the single source of truth in Director OS.

The entire AI director system never modifies prompts directly.

All Agents are only allowed to modify Project.

All Prompts are auto-generated by Compiler from Project.

Workflow:

User Idea
    →
Creative Brief
    →
Project
    →
Story Bible
    →
Shot Bible
    →
Compiler
    →
Prompt

Project describes "the film itself".

It is not the Prompt.

---

# 2. Core Principles

Project must satisfy the following principles:

## 2.1 Single Source of Truth

All content can only exist in one place.

For example:

Character wardrobe

Only recorded once.

Must not be duplicated across prompts.

---

## 2.2 Structured

Project must use exclusively structured data.

No long-form free text Prompts.

For example:

✓

camera:
    lens: 35mm

✗

Camera uses a cinematic 35mm lens...

---

## 2.3 Immutable History

Every change must be recorded.

Project is always rollback-able.

---

## 2.4 Model Independent

Project must never contain:

Seedance

Veo

Kling

Runway

Those belong to Compiler.

Project stays neutral.

# 13. Shot List

Shot grammar is defined in full by [SHOT_GRAMMAR.md](SHOT_GRAMMAR.md).

The canonical Shot shape:

shots:

-

    shot_id:            # unique identifier
    beat_ref:           # references story_beats[].beat
    order:              # shot sequence number
    duration:           # seconds (float)

    subject:
        character:      # references characters[].id
        action:         # what the subject is doing
        position:       # position in frame
        state:          # standing / sitting / lying

    camera:
        framing:        # ECU / CU / MCU / MS / MLS / LS / ELS / EST / OTS / POV
        angle:          # EYE_LEVEL / HIGH_ANGLE / LOW_ANGLE / BIRDS_EYE / DUTCH_ANGLE
        height:         # GROUND / WAIST / CHEST / EYE / HIGH
        lens:           # 14mm / 24mm / 35mm / 50mm / 85mm / 135mm / 200mm
        movement:       # STATIC / HANDHELD / DOLLY_IN / TRACK / CRANE / ...
        focus:          # DEEP_FOCUS / SHALLOW_FOCUS / RACK_FOCUS / SOFT_FOCUS
        aperture:       # f1.4 / f2.8 / f5.6 / f8 / ...

    composition:
        rule:           # RULE_OF_THIRDS / CENTER / SYMMETRY / LEADING_LINES / ...
        depth:          # number of depth layers (2-5)
        foreground:     # what's in front
        midground:      # middle layer
        background:     # what's behind

    lighting:
        key_light:      # NATURAL / HARD / SOFT / MOTIVATED / ...
        position:       # FRONT / SIDE / BACK / TOP / UNDER / ...
        intensity:      # 1-10
        temperature:    # 2000K / 3200K / 5500K / 6500K / ...
        mood:           # emotional description

    color:
        palette:        # dominant colors
        accent:         # accent color
        contrast:       # contrast level

    audio:
        silence:        # true / false
        dialogue:       # dialogue content
        ambience:       # ambient sound
        sound_effects:  # SFX
        music:          # music cue

    transition_in:      # CUT_TO / FADE_TO / DISSOLVE_TO / SMASH_CUT_TO / ...
    transition_out:     # CUT_TO / FADE_TO / DISSOLVE_TO / MATCH_CUT_TO / ...

    emotion:
        target:         # target emotional state
        intensity:      # 1-10

    notes:              # director's notes

Note:

Shot is director language.

It is not a prompt.

# 20. Schema Governance

This document (PROJECT_SCHEMA.md, v2.0.0) is the canonical schema for
Director OS project files. All other representations are either:

  - Runtime artifacts (Python dataclasses in director_os/models/project.py)
  - Field-level references (SHOT_GRAMMAR.md for Shot substructure)
  - Deprecated drafts (schemas/project.schema.yaml)

The 14-module flat layout described above is what the code, tests, and
all project files in projects/ adhere to. Any future schema changes must
be reflected here first.

# 21. Design Philosophy

Director OS does not generate prompts.

Director OS creates films.

Prompts are only the final export format of a film.

Project is the film.

Prompt is only the film's PDF.

Project is the true source file.
