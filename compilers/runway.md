# Director OS — Runway Compiler

Platform: Runway ML
Status: Planned
Version: 0.1.0

## Overview

Runway Compiler 将 Director OS Project 编译为 Runway Gen 系列可用的 Prompt。

## Runway-Specific Rules

- Frame-by-frame description style recommended
- Visual style and aesthetic keywords emphasized
- Camera movement described as continuous motion
- Color grading and atmosphere explicitly stated

## Translation Map

| Shot Grammar       | Runway Prompt Pattern                      |
|--------------------|--------------------------------------------|
| framing: MS+action | "Medium shot of [subject] [action]"        |
| movement: HANDHELD | "Handheld camera, slight shake"            |
| color: palette     | "Color palette: [description]"             |
| atmosphere: mood   | "[Mood] atmosphere with [lighting]"        |

## Strengths

- 对美学风格描述敏感,适合艺术化表达
- 支持多种风格迁移
- 镜头时长控制精确

## Constraints

- 描述尽量具体,避免模糊词汇
- 角色一致性需要持续强化描述
- 运动镜头需要明确 start/end 状态
