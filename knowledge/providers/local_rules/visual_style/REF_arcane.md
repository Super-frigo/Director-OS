# Arcane 风格拆解 (Original Reference)

> Imported from AI-Director-OS/stylebook/arcane.md

## 视觉 DNA

### 手绘质感
- 非真实渲染（NPR）+ 手绘贴图叠加
- 轮廓线随光影变化粗细
- 烟雾、火焰、魔法效果采用 2D 逐帧手绘

### 油画笔触
- 大色块铺陈 + 可见笔触方向
- 阴影区域使用交叉排线
- 高光区域保留笔触留白

### 戏剧光影
- 强烈的明暗对比（Chiaroscuro）
- 面部打光精确到角色情绪
- 霓虹/彩色光斑融入手绘表面

## 色彩体系

### 双城色彩对立
- 皮尔特沃夫：金 + 白 + 淡蓝
- 祖安：绿 + 紫 + 锈红

### 通用规则
- 70% 暗部 + 30% 戏剧性亮部
- 色彩随角色命运弧线变化
