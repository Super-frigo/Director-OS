# Color Provider

**Type:** Local Rules
**Deterministic:** Yes
**Source:** YAML entries in this directory

## Purpose

Provides structured knowledge about color psychology:
- Color-emotion mappings (red/passion, blue/cold, yellow/warmth)
- Palette strategies (complementary, analogous, monochromatic)
- Desaturation effects
- Teal-orange and other cinematic color schemes

## Resolution Strategy

Direct keyword + emotion matching against entry fields.

## When LLM Extension Is Needed

- Director-specific color signatures ("Wes Anderson pastels", "Fincher desaturation")
- Cultural color symbolism (Chinese red, Western mourning black, etc.)
- Complex palette combinations for mixed emotions
