# Storytelling Provider

**Type:** Local Rules
**Deterministic:** Yes
**Source:** YAML entries in this directory

- Story beat types (16 beat types from STORY_GRAMMAR)

- Non-Western narrative structures (kishotenketsu, circular narratives)
- Genre-blending story templates
- Experimental narrative forms
